# Frontend

## Setup

```bash
npm install
npm run dev
```

Create `.env.local`:

```
NEXT_PUBLIC_API_BASE_URL=http://localhost:8080
NEXT_PUBLIC_FAKE_TASK_POLL=true
```

## Features

- Suche mit Facetten und Export
- Detailansicht
- Mehrdatei-Import mit Upload-Progress
